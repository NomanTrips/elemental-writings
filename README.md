# elemental-writings
Simple writing software for the internet
# directory structure
elemental-writings/  
├── content/  
│   └── example-post.md  
├── public/  
│   ├── styles.css  
│   └── script.js  
├── server.js  
├── package.json  
└── README.md
# installation
- git clone https://github.com/NomanTrips/elemental-writings.git
- cd elemental-writings
- npm install
- npm start
# access the application
- open your browser and navigate to http://localhost:3000
